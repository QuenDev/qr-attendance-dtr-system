Help & Support

📖 Getting Started

  • Add members in the Members section
  • Each member has a QR code for Time In/Out
  • Use the Employee Scanner to capture QR events
  • View live Time-Ins and attendance logs in the Dashboard

⚙️ Common Tasks

  • Filter attendance by week/month/year or custom range
  • Print attendance (daily/weekly/monthly/yearly) using logbook format
  • Export CSV reports from the Reports section

🔍 Status & Logbook

  • Attendance Log shows raw AM/PM Time In/Out entries
  • Dashboard shows real-time Time In or Time Out status
  • Notifications and Logs capture admin actions and QR events

🚀 Tips

  • Use the date range filters to narrow results quickly
  • Refresh the Dashboard for latest live stats
  • Regularly back up data from Settings

❓ FAQs

  • Q: How to change admin password?
    A: Settings → Admin Accounts → Change Password
  • Q: Can members time in twice a day?
    A: Yes — AM and PM sessions
  • Q: Where to print attendance?
    A: Attendance Log → Print buttons

📞 Support & Feedback

For issues or feature requests, use system logs to gather details, then contact support.
